## 🎨 Concepto

Un sitio web que refleja creatividad y dinamismo, enfocado en destacar proyectos visuales.

## ✨ Features

- Animaciones suaves tipo “magnetic UI”  
- CMS para edición de contenido  
- Diseño responsive  

## 🎯 Objetivo

Mostrar el portafolio de forma atractiva y aumentar conversiones.

## 🛠 Stack

Astro para performance, Svelte para interacciones y Sanity como CMS.
