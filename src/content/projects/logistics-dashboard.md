## 📦 Sobre el proyecto

Plataforma SaaS para empresas de transporte que necesitan visibilidad total sobre sus operaciones.

## ⚙️ Funcionalidades

- Seguimiento de rutas en tiempo real  
- Gestión de inventario  
- Reportes automáticos  
- Panel de analíticas  

## 📈 Impacto

Reducción del 35% en tiempos de entrega para clientes.

## 🧩 Arquitectura

Sistema distribuido con APIs desacopladas y base de datos optimizada para consultas complejas.
